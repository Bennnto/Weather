# Weather
## Description : <br>
   ##### Weather Application write in Python with requests and PyQt5 libraries.<br>
   ##### this application use information from Openweather V3.0 to use this application <br>
   ##### Register account and get API_KEY from [Click Here](https://home.openweathermap.org/users/sign_in) <br>
   ##### After you get API_KEY copy and paste in at the top of the application file <br>
   ##### this application ask user to input city (city name eg. Bangkok, Toronto, Tokyo etc..) after that will call <br>
   ##### API to Openweather Geocoding API to get the return of latitude and longitude of that city and will pass it to <br>
   ##### call_info function to call Openweather V3.0 API and get return weather dataset in json after that call_info will <br>
   ##### extract information from json file and modified instance of PyQt5 elements to show data in term of Graphic User Interface <br>
